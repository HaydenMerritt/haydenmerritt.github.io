3D Models/3D Model Projects

This page is dedicated to featuring all projects/models contributed/produced by me.

Forest Project

The forest project was produced to help me explore more in Blender cycles to improve on my lighting and techniques in the program, this was quite a success as you can see. However as I developed it further I realised additional features I could potentially use in the future, for example Volume Scatter for the environment in future projects.

Stone Axe

The stone axe was produced to help me further understand how Normal Maps and UV Maps worked, additionally it taught me how to use a reference sheet to develop the model

My 3D modelling experience

3D modelling helped me in general understand more about how games worked, how rendering worked in terms of faces on the object. Trying different lighting angles, brightness and techniques to achieve a better idea. The smaller models I've made had provided to be ideal assets to use during my games.

I'm currently able to use Blender and produce industry-standard models. Additionally I know how to produce textures using Photoshop and combine shaders within Blender cycles.

What I wish to achieve next.

Acknowledging I've been using Blender for a fair amount there is still a lot I wish to achieve with the skillset I currently have.

  • Further improve my 3D modelling efficiency.
  • Learn a new 3D modelling program like Maya, or 3DS Max.
  • Produce an animation or an animated asset.
  • Improve my UV mapping and normal mapping.
  • Create a whole atmosphere using Blender Cycles or a similar rendering engine.
